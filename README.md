# AI Customer Support Chatbot 🤖💬

A Python chatbot that handles customer service queries using AI with full conversation history.

## Features
- Answers customer questions based on business information
- Maintains full conversation history for context
- Handles products, pricing, shipping, returns and payment queries
- Falls back gracefully when information is not available
- Built with LLaMA 3.3 70B via Groq API

## Tech Stack
Python, Groq API, LLaMA 3.3 70B

## How to Run
1. Install dependencies: `pip install groq`
2. Add your Groq API key in the configuration section
3. Run: `python main.py`
4. Type your questions in the terminal

## Example Use Case
Deployed for TechShop Sofia — an online electronics store. Handles customer queries about products, pricing, shipping and returns automatically without human intervention.
